Gatee
========================

#####Compilation commands:
Initially compile Gatee files by using the following command.

```
$ make gatee
```

If you change the source files, you can 'clean' the build related files with:

```
$ make clean
```

#####Running Gatee:

Use the following command to run Gatee:

```
$ . runGatee
```

#####Exiting Gatee:

Press FN + F1 to exit.
